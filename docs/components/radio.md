# Radio 单选按钮

单选按钮组件用于在多个选项中选择一个，通常与 RadioGroup 组合使用。

## 基础用法

:::radio/basic

## 内联布局

水平排列的单选按钮，适合选项较少的场景。

:::radio/inline

## 不同状态

包括正常、禁用、只读等状态。

:::radio/states

## 颜色主题

支持多种预设颜色主题。

:::radio/colors

## 样式变体

提供默认和边框等多种样式变体。

:::radio/variants

## 自定义图标

支持自定义选中和未选中状态的图标。

:::radio/icons
