# Drawer 抽屉

SPDrawer 抽屉组件是一个从屏幕边缘滑出的面板，用于展示额外的内容或操作。常用于导航菜单、筛选面板、详情展示等场景。

## 基础用法

最基本的抽屉用法，从右侧滑入。

:::drawer/basic

## 不同位置

抽屉可以从四个方向滑出：左侧、右侧、顶部、底部。

:::drawer/location

## 自定义宽度

通过 width 属性可以设置不同的抽屉宽度。

:::drawer/width

## 激活器插槽

使用 activator 插槽提供更灵活的触发方式。

:::drawer/activator
