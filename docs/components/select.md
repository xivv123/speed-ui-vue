# Select 选择器

选择器组件允许用户从预定义的选项列表中进行选择，支持单选和多选模式。

## 基础用法

基本的选择器功能，提供多种变体样式。

## 清除功能

带有清除功能的选择器，选择后可以清空。

:::select/clear

## 多选模式

多选模式的选择器，可以选择多个选项。

:::select/multiple

## 图标配置

带有图标和前后缀的选择器示例。

:::select/icon

## 不同尺寸

支持不同的尺寸和密度设置。

:::select/sizes

## 不同状态

:::select/validation
